2020-04-16

RStudio update
Thursday, April 16, 2020

RStudio server was upgraded to the newest version (1.2.5033-1). Multiple R-versions are now supported and r-sessions start in a clean environment.

Rstudio

  • Update to version RStudio Server Pro 1.2.5033-1
  • Rstudio session will no longer read ~/.bashrc or ~/.bash_profile.
  • Support for multiple versions of R (currently 3.5.1 and 3.6.0, bare or with texlive+bioconductor)
  • The default R version can be changed by the user and is remembered when restoring an R project
