About CLIP

The Concept.

The Cloud Infrastructure Platform (CLIP) consists of heterogeneous computing hardware. It is entirely virtualized using OpenStack, thus enabling rapid recondiguration and seamless extensibility.

For fast intermediate storage of computational data, CLIP features an all-flash BeeGFS cluster filesystem 250 TB in size. Data on this scratch space has a limited lifetime of 30 days and thus must be staged in and out of the system as needed.

Software Environment

CLIP nodes run a streamlined CentOS 7 operating system. For all computationally relevant software, CLIP/CBE features the Lmod hierarchical module system. Lmod software modules can be loaded on demand to assemble an environment with the software versions and optimisations required for the task at hand. Modules will automatically pull in other modules they depend on, and one can save sets of frequently used modules as Collections.

How to cite us

If you would like to acknowledge CLIP and its services in your publications, we suggest the following sentence and, if possible, link. We will provide a DOI pointing to this site shortly.

The computational results presented were obtained using the CLIP cluster.
